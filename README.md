## Edit The Config

Edit `config.json` to customize the stickers
```JSON

{
    "name": "KYOMI-Bot:",
    "author": "Made by: ",
    "timezone": "   ",
    "groups": true
}

```

##  Installation
* Clone Repository: 
```bash

>git clone https://github.com/Alpha-Dev0/Kyomi-BOT/

```
* Install Modules
```bash

> npm install

```

* Starting the bot

```bash

> npm start

```


* Features
